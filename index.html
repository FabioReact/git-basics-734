Les bases de Git

git init

Cette commande permet d'initialiser git sur le projet voulu

git status

Nous donne le status des fichiers:

  • untracked (non suivi): le fichier n'est pas suivi par git
  • added (ajouté): le fichier vient d'être ajouté à la zone de suivi
  • modifié (not staged): le fichier est modifié, mais ces modifications ce feront pas parties de la prochaine sauvegarde

git add monFichier

Cette commande ajoutte monFichier à la zone de suivi
